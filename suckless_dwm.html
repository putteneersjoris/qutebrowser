dwm shortcuts
new terminal = alt ctrl enter
alt d = dmenu
alt 1 = workspace 1
alt j/alt k = prev tag
shift alt c = quit tag
shift alt q = quit dwm
alt h / alt l= move tag to the side
alt b = remove workspace bar


change terminal style ~$ sudo nano ./config/duckless/config.h
~$ sudo make clean install


alter settings:
~$ sudo nano Documents/dwm/config.h
~$ sudo make clean install
